This one you need to run to set the polices of the tables, to only authenticated user have access

```roomsql
-- For storage_items table
ALTER TABLE storage_items ENABLE ROW LEVEL SECURITY;
ALTER TABLE storage_items FORCE ROW LEVEL SECURITY;
DROP POLICY IF EXISTS "Authenticated users can do everything" ON storage_items;
CREATE POLICY "Authenticated users can do everything"
  ON storage_items
  TO authenticated;
REVOKE ALL ON storage_items FROM PUBLIC;
```
